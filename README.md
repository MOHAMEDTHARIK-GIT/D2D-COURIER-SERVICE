📌 Project Overview

The Door-to-Door Courier System is a web-based application designed to automate and manage courier and delivery services efficiently.

In today’s digital world, courier services play a vital role in supporting e-commerce businesses, retail stores, pharmacies, and other service providers. Manual courier management often leads to delays, miscommunication, and data management issues.

This project provides a centralized and automated platform that simplifies parcel booking, tracking, and delivery management.


🎯 Objectives

   To automate the courier booking process

   To manage parcel details efficiently

   To track delivery status in real-time

   To reduce manual errors

   To improve customer satisfaction


👥 Users of the System

1.Admin

  Manage users and delivery agents

  View all bookings

  Update parcel status

  Delivery Agent

  View assigned deliveries

  Update delivery status

  Customer

  Book a courier

  Track parcel status

  View booking history
  

🛠️ Technologies Used

    Frontend: HTML, CSS, JavaScript

    Backend: PHP

    Database: MySQL

    Server: XAMPP / Localhost
    

⚙️ System Features

  User Registration and Login

  Courier Booking System
  
  Parcel Tracking

  Admin Dashboard

  Delivery Status Update

  Secure Database Management

💻 How to Run the Project

  Install XAMPP on your system

  Copy the project folder into the htdocs directory

  Start Apache and MySQL in XAMPP

  Import the database file into phpMyAdmin

  Open your browser and go to:

    http://localhost/project-folder-name
